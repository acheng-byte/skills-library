# 新技能模板

按照以下格式创建你的技能文件（SKILL.md）：

```
---
name: your-skill-name
description: "描述这个技能做什么，以及什么时候触发它。包含触发词示例。"
---

你是[角色定义]，专门帮助用户[核心任务]。

## 核心功能

[描述这个技能的主要功能]

## 工作流程

1. 收到[输入类型]后，执行：
   - [步骤1]
   - [步骤2]
   - [步骤3]

2. 输出格式：
   - [格式说明]

## 示例

**用户输入**：[示例输入]

**技能输出**：[示例输出]

## 注意事项

- [限制或注意点1]
- [限制或注意点2]
```

---

## 字段说明

| 字段 | 要求 |
|------|------|
| `name` | 英文小写，用连字符，简短有意义（如 `vc-copilot`） |
| `description` | 说清楚：做什么 + 什么时候触发。包含关键词让AI知道何时调用 |
| 指令内容 | 你的AI角色设定 + 工作流程 + 输出格式 |

## 好的 description 示例

```
"VC 投资分析工具。当用户提到 BP、项目分析、赛道研究、研报压缩时使用。"
"费曼学习法教练。帮助深度理解新概念，用于学习和备考。"
"Word文档生成。当用户需要创建.docx文件时使用，通过JavaScript生成。"
```
