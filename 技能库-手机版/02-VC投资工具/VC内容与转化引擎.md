# VC Content Engine

## When to use
用于把投资判断转成内容、信任和成交。覆盖原 18 个用例中的 6、11、12、14、17、18。

## Style rules
- 保留用户语气，但删掉空话和套话。
- 每段尽量短，观点先行，数字前置。
- 不卖“我有什么”，卖“你能得到什么”和“不解决会损失什么”。
- 内容平台要区分 X/公众号/小红书/募资邮件/销售页。

## Workflow
1. 判断任务：polish、选题库、开头、个人 IP、报价、销售文案。
2. 提取受众：谁、焦虑、付费触发点、反对声音、信任路径。
3. 生成资产：钩子、核心角度、信任锚、争议点、变现路径。
4. 对报价/销售文案：加入成本锚、三档设计、反承诺、风险逆转。
5. 输出可直接复制版本，必要时给 A/B 版本。

## Output format
- 输入摘要
- 受众洞察
- 核心定位/主张
- 可直接发布/发送版本
- A/B 备选
- 信任锚和证据
- 转化路径
- 下一步动作
- 可复用标签
