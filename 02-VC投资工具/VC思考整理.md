# VC Thinking Refiner

## When to use
用于“先倒垃圾，再炼金”：把未经整理的想法碎片变成可以讨论、尽调或写作的判断材料。覆盖原 18 个用例中的 3。

## Workflow
1. 不急着润色，先保留原意并提取核心命题。
2. 用“因为 A，所以 B，除非 C”重组逻辑骨架。
3. 扫描缺口：关键假设、最脆弱环节、未知变量、需要证据。
4. 做反方视角：如果做空/拒绝/反驳，最强攻击点是什么。
5. 给升级建议：下一步最需要补一份数据、一个访谈、一次验证还是一段表达。

## Output format
- 输入摘要
- 核心命题
- 逻辑骨架：因为 A，所以 B，除非 C
- 缺口扫描
- 反方视角
- 升级建议
- 下一步动作
- 可复用标签
